Plis_Bsk_v3_01 
Апрель 2018

Прошивка ПЛИС Prd дял платы АВАНТ BSK_023 (DD7 Altera EPM7128STC). Написана на Verilog, за основу взята прошивка v2.5.

Для блока БСК 01-16 адрес CS = 1011 (все тесты сделаны для него).
Для блока БСК 17-32 адрес CS = 1001. 

Quartus 13.0 sp1
python 2.7.14
vunit-hdl (3.0.2)

"run.py" - запуск VUnit тестов с выводом информации в консоль.
"run.bat" - запуск VUnit тестов в отладчике (ModelSim).
